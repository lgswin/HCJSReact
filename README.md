## VS code short key
* formatter (prettier) : preferences -> settings -> "formatter" -> default formatter -> put prettier as default
=> short key : ⌥⇧

* close/open left side window : ⌘b

* markdown preview open : ⌘⇧v

* split window : ⌘\

* move windows : ⌘⇧[, ⌘⇧]
